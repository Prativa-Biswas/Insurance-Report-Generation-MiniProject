# Insurance Report Generation

## Project Description

Insurance Report Generation is a Spring Boot application that allows users to search insurance plan details using different filters such as plan name, plan status, start date, and end date.
The application generates reports based on the search results and sends the generated report to the user's email as an attachment.

## Technologies Used

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA
* MySQL Database
* Maven
* Jasper Reports
* Spring Boot Mail
* HTML, CSS, Bootstrap

## Features

* Search insurance plans using multiple filters
* Generate reports based on filtered data
* Export reports in **PDF / Excel format**
* Automatically send the generated report to the user's **email**
* Store and retrieve insurance plan data from the database

## Project Structure

```
src/main/java       → Application source code
src/main/resources  → Configuration files
src/test/java       → Unit test classes
pom.xml             → Maven dependencies
```

## How to Run the Project

1. Clone the repository
2. Open the project in Eclipse or STS
3. Configure database details in `application.properties`
4. Configure email settings for sending reports
5. Run the Spring Boot application
6. Use the application to generate reports and receive them via email
